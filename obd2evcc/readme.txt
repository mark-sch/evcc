Install (requires node v14+):
================================
> npm Install


Getting started:
================================
Discover obd dongles by wifi and create config file:
> node build-smartplug-config.js

Adjust individual settings in: config.js

Run service (manually):
> node smartplug-service.js



Peugeot OBD Pinout:

3 zu 6
4 zu 4
8 zu 14
16 zu 16

Es sind nur vier Verbindungen.
